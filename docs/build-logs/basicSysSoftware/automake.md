# Package: Automake

**Version:** 1.18.1
**Chapter:** 8.48 (System Software)
**SBU Estimate:** < 0.1 SBU (1.1 SBU with tests)

## Overview
The Automake package provides the companion tool to Autoconf. It automatically generates standardized `Makefile.in` files from user-defined `Makefile.am` files. By scanning the configuration files generated by Autoconf, it seamlessly creates the final Makefiles required to compile source code across varying Unix-like architectures.

Key binaries included:
* `automake` (The core Makefile generator)
* `aclocal` (Generates `aclocal.m4` macro files based on Autoconf configure files)
* `automake-1.18` & `aclocal-1.18` (Hard links to the primary binaries to preserve version-specific build script compatibility)

## Security / Custom Modifications
- **Strict FHS Layout:** Configured with `--prefix=/usr` to ensure the toolchain integrates perfectly with the root filesystem execution path.
- **Documentation Isolation:** Explicitly passed `--docdir=/usr/share/doc/automake-1.18.1` to cleanly sandbox the extensive Automake documentation by version, preventing namespace collisions and maintaining filesystem hierarchy integrity.

## Build Notes
- **Required Disk Space:** 123 MB
- **Process:** Compiled cleanly in-tree using standard configurations.
- **Test Suite:** Executed the test suite using a dynamic parallel job calculation (`make -j$(($(nproc)>4?$(nproc):4)) check`). This specific logic forces the test suite to use at least four parallel jobs (even on systems with fewer logical cores) to heavily optimize the 1.1 SBU test time, compensating for internal I/O delays hardcoded into individual tests.
- **Installation:** Standard `make install` execution.
