# Package: Patch (Temporary Native Tool)

**Version:** 2.8
**Chapter:** 6 (Cross-Compiling Temporary Tools)
**SBU Estimate:** 0.1 SBU

## Overview
The Patch utility modifies files by applying differences generated by `diff`. It is a strict prerequisite for the target sysroot, as several upcoming packages require upstream source code patches to compile correctly within the chroot environment.

## Security / Custom Modifications
- **Target Sysroot Mapping:** Configured explicitly with `--host=$LFS_TGT` to utilize the cross-toolchain.
- **Cleanroom Installation:** Installed via `make DESTDIR=$LFS install` to correctly map the binary into the target's `/usr/bin` directory, avoiding any contamination of the host's `/usr` tree.
