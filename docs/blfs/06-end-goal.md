it throws out a hardening index score out of 100, which determines the security of my lfs system.

an out-of-the-box ubuntu or debian server usually scores around a 60-65. because lfs has zero default security configs, a raw lfs build will likely score in the 30s.

my goal is to get 80+ out of 100, as i don't have a broad knowledge of security as of now, but i will try to build and learn as i go, since i find it easier to learn by building and making stuff.
